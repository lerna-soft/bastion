# HIM-003: Auditoría funcional + Server Info Panel + Sidebar colapsable

## Feature
Mejora integral de UI: reemplazar elementos no funcionales, agregar panel de información del servidor en terminal, y sidebar colapsable.

## Prioridad
Alta

## Estimación
8 puntos

## Estado
Borrador

---

## Historias de usuario

### HU-1: Server Info Panel
**Como** operador de infraestructura  
**Quiero** ver información del servidor cuando abro una terminal  
**Para** saber a qué servidor estoy conectado, su IP, puerto, estado de la conexión, y poder expandir esa información para ver detalles

### HU-2: Sidebar colapsable
**Como** operador de infraestructura  
**Quiero** poder colapsar el sidebar a iconos  
**Para** tener más espacio horizontal para la terminal y servidores

### HU-3: Reemplazar elementos no funcionales
**Como** usuario  
**Quiero** que todos los botones y enlaces tengan una acción útil  
**Para** no encontrar elementos "muertos" en la interfaz

---

## Escenarios Gherkin

### Escenario HU-1: Server Info Panel visible al conectar
```gherkin
Dado que tengo una conexión SSH activa
Cuando la terminal se abre
Entonces veo un panel con el nombre del servidor, hostname, puerto y estado
Y el panel está colapsado por defecto
```

### Escenario HU-1: Expandir server info
```gherkin
Dado que estoy viendo una terminal activa
Cuando toco el panel de información del servidor
Entonces el panel se expande mostrando: hostname, IP, puerto, username, auth type, tiempo de conexión, versión del servidor
```

### Escenario HU-2: Colapsar sidebar
```gherkin
Dado que estoy en la pantalla principal
Cuando toco el botón de colapsar en el sidebar
Entonces el sidebar se reduce a solo iconos (≈64px)
Y los labels de navegación se ocultan
Y el contenido principal ocupa el espacio liberado
```

### Escenario HU-2: Expandir sidebar
```gherkin
Dado que el sidebar está colapsado
Cuando toco el botón de expandir
Entonces el sidebar vuelve a su tamaño completo (260px)
```

### Escenario HU-3: Elementos funcionales
```gherkin
Dado que navego por la aplicación
Cuando interactúo con cualquier botón o enlace
Entonces siempre ocurre una acción visible
Y ningún elemento muestra solo un toast sin funcionalidad real
```
